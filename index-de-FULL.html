Was ist das Pi-Netzwerk?

Pi Network ist ein neues Kryptowährungsprojekt, das es Ihnen ermöglicht, Münzen direkt von Ihrem Mobilgerät aus zu schürfen, ohne den Akku zu entladen oder technisches Wissen zu benötigen. Es wurde von Stanford-Absolventen entwickelt und hat bereits Millionen von Nutzern.

Wie es funktioniert

Pi zu schürfen ist so einfach wie einmal täglich auf einen Button zu tippen. Alle 24 Stunden beginnen Sie eine neue Mining-Sitzung. Je mehr Personen sich mit Ihrem Code anmelden, desto schneller schürfen Sie. Keine komplizierte Einrichtung, keine teure Hardware.

Häufige Fragen

Ist Pi eine echte Kryptowährung?
Ja. Pi basiert auf einer eigenen Blockchain und wird handelbar sein, sobald das Mainnet vollständig geöffnet ist.

Ist das Mining von Pi sicher?
Vollständig. Die App greift nicht auf Ihre persönlichen Daten zu und läuft nicht im Hintergrund, wodurch der Akku geschont wird.
